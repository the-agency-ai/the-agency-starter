## Usage Guidelines

### Text Colors

| Purpose | Token | Example |
|---------|-------|---------|
| Primary text | `grey-900` or `black` | Body copy, headings |
| Secondary text | `grey-700` | Subdued content |
| Disabled text | `grey-500` | Inactive elements |
| Inverse text | `white` | On dark backgrounds |

### Background Colors

| Purpose | Token |
|---------|-------|
| Page background | `off-white` or `grey-100` |
| Surface (cards) | `white` |
| Muted surface | `grey-100` |
| Dark surface | `grey-900` |

### Border Colors

| Purpose | Token |
|---------|-------|
| Default border | `grey-300` |
| Strong border | `grey-500` |
| Focus ring | `primary-500` |

## Accessibility Notes

- Ensure 4.5:1 contrast ratio for body text
- Ensure 3:1 contrast ratio for large text and UI components
- Test semantic colors for color blindness accessibility
